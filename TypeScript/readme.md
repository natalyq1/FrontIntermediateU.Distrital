# Typescript

## Generación js
Para generar el código Javascrip a partir del código typescript, se debe ejecutar el siguiente comando:

        tsc -w
El código fuente(ts) se encuentra en `src/ts` y el código resultante (js) se encuentra en `dist/js`.
